# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

Zenvy Tech is a simple static website for online orders. We take security seriously.

If you discover a security vulnerability, please report it privately to us.

**Please do NOT report security vulnerabilities through public GitHub issues.**

Instead, please contact us directly:

- **WhatsApp:** 923012800023
- **Email:** security@zenvytech.com

**Please include:**
1. Description of the vulnerability
2. Steps to reproduce the issue
3. Potential impact

We will respond within 48 hours and work with you to resolve the issue.

Thank you for helping keep Zenvy Tech safe!
